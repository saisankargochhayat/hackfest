OVERVIEW

Tensorflow Object Detection Tutorial

This workshop mainly focuses on the students eager to learn Machine Learning from Basic. They will get the chance to expand their knowledge in the field of designing, construction, operation, and application of Machine Learning with real time hand on practical experience. The duration of this workshop will be 1 day , with 3 Sessions , 2.5 hours each, properly divided into theory and hand on practical sessions.
Best Suited for : All B.Tech/B.E./BCA/BSc. Students

SESSION 1
  • Introduction to Machine Learning and Neural networks
  • Linear regression in details
  • Training the neural network using linear regression for image classification
SESSION 2
  • Discussion on need of custom object detection
  • Object detection tutorial using tensorflow object detection api
  • Real time detection using cam feed
SESSION 3
  1. Discussion on need of custom object detection
  2. Train your custom object detection Model for specific application
